# CTeSP-IPVC Website

## Description
This project simulates the official CTeSP website, focusing on web and multimedia development at the Polytechnic Institute of Viana do Castelo.

## Technologies
This is a static website built using:
- HTML
- CSS
- JavaScript

It also uses external fonts and includes a favicon for improved visual identity.

The project includes basic web development concepts such as forms, tables, links, and image slideshows.
